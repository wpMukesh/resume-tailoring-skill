Experience

Company: Webspruce

Project: Note For Approval Workflow System, A approval workflow system used in manufacturing industry where users create Note For Approval requests from reusable templates, fill dynamic request details, where they clarify and get approvals. The system used Hono, PostgreSQL, Drizzle ORM, Zod, React, Vite, TanStack Query, TanStack Table, React Hook Form, Tailwind, and dnd-kit.

- Built a NFA workflow product covering template-based request creation, dynamic request details, role-based member management, clarifications, and approval routing, with strict TypeScript, linting, formatting, pre-commit hooks, and build checks maintained across frontend and backend repositories.

- Designed an extensible PostgreSQL schema for a template-based approval workflow, supporting admin-created templates, dynamic request details, approver/viewer/clarifier members, clarifications, posts, and edit history.

- Standardized API response envelopes for success, error, warning, and no-op cases, allowing the frontend to consistently handle typed data, server-driven user-visible warnings, validation failures, and unchanged updates.

- Built composable Zod primitives, schema factories, typed API wrappers, and normalized React Query key factories to keep backend APIs, frontend clients, RHF validation, cache reads, and invalidations type-safe and consistent.

- Designed reusable list API patterns with search, typed date filters, typed sort columns, cursor pagination, and hydrated listing responses, improving consistency across modules and reducing extra frontend data-fetching.

- Built a template-configured request form system where admins creates fields such as rich text, options, and numeric inputs, configures template using these fields. As well as an intiutive responsive UIs for field creation, template configuration, and request details submission.

- Implemented server-authoritative, configuration-driven validation for submitted request details, enforcing option membership, template configuration like required fields, selected fields, and more.

- Built role-based request member workflows for approvers, viewers, and clarifiers, supporting drag-and-drop approver ordering with atomic reorder updates.

- Built reusable frontend data-table components with configurable debounced search, selectable date filters, type-safe column sorting, loading states, and cursor-based pagination with previously visited page navigation.
